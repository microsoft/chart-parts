Optional keyRowName

keyRowName: undefined | string

The name to use for the group mark's bound-data row in the context of facet encoding. If the group mark has a bound data row, it will be mapped to a facet partition using the facet's group-by mechanism. If the group is not bound to a data row, it will be bound to the data facet partitions.
